# DeviceDetection
Small project for identifying CPU(s) features and OpenCL (AMD) and CUDA (NVIDIA) GPUs.

## Get started
``` 
git clone git@github.com:nicehash/DeviceDetection.git
cd DeviceDetection
git submodule update --init --recursive
```
